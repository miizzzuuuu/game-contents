Support Languages:

-   Hindi (hi)
-   Indonesian (id)
-   Korean (ko)
-   Brasilian Portuguese (pt-BR)
-   Thai (th)
-   Turkish (tr)
-   Vietnamese (vn)
-   Chinese (zh)

## Mini How To Play

Access the folder: `games/{game_name}/locales/{language}/mini-htp.json`

Example:

-   Game: `12D`
    -   Language: `English (en)`
    -   File: `games/12d/locales/en/mini-htp.json`
